# 一路通关Python全栈开发
这个仓库文档用于全栈开发学习，本仓库将从Vue3一直到Django5的学习，来速通一遍基础常用知识，之后再通过一个商城项目进行一个巩固，最后再学习Linux上项目的部署与运行，最大程度地完整教学开发一个全站项目的全部流程。
